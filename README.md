# 🌐 Small Office Home Office (SOHO) Network Design & Implementation

## Overview

This project involved designing and implementing a Small Office Home Office (SOHO) network using Cisco Packet Tracer.

The objective was to create a scalable and segmented network supporting multiple departments while enabling controlled communication and automated IP allocation.

---

## Project Requirements

✔ One Router
✔ One Switch
✔ Three Departments
✔ Separate VLAN segmentation
✔ Wireless access support
✔ Automatic IP allocation (DHCP)
✔ Inter-department communication

---

## Network Architecture

Departments:

### VLAN 10 — Administration

Network: 192.168.1.0/26

### VLAN 20 — Finance

Network: 192.168.1.64/26

### VLAN 30 — Reception

Network: 192.168.1.128/26
